```
 font-family : 웹 문서의 글꼴 지정

 font-size : 글자의 크기를 지정

 font-style : 글자를 이텔릭체로 표시하는 속성

 font-weight : 글자 굵기를 조절

text-align : 텍스트를 정렬하는데 사용

line-height : 문단의 줄 간격을 지정

text-decoration :텍스트에 밑줄을 긋거나, 텍스트 링크의 밑줄을 지울 때도 사용

text-transform : 영문 텍스트의 대문자나 소문자를 바꾸는 속성

list-style-type : 순서 없는 목록의 불릿이나 순서 목록의 숫자를 바꾸는 속성 ( ul, li 등에 사용)

list-style-image : 순서 없는 목록의 불릿을 이미지로 바꾸는 속성

list-style-position : 불릿이나 번호를 들여쓰거나 내어쓸 수 있음

width, height : 넓이, 높이를 조정

box-sizing : 실제 박스 모델의 너비를 계산할 때 어디까지 포함할지를 결정하는 속성

● 박스의 영역 : content < padding < padding <border < margin으로 나뉨

border : 테두리의 영역을 선과 같은 것으로 표시 ex) border : 2px solid black;

border-radius : 박스 모델의 테두리를 둥글게 처리

display : 요소의 배치 방법을 결정

float : 요소를 왼족이나 오른쪽에 떠 있게 만듬

clear : float 속성을 무효화 시키는 속성

left, right, bottom, top : 웹 요소의 위치를 지정하는 속성

position : 웹 문서 안에 요소들을 배치하기 위한 속성
1)static : 요소를 문서의 흐름에 맞추어 배치
2)relative : 이전 요소에 자연스럽게 연결해 배치하되 위치롤 조정가능
3)absolute : 원하는 위치를 지정해서 배치
4)fixed : 지정한 위치에 고정 배치

등등 css에는 더 많고 다양한 속성들이 있다. 나머지는 실습을 활용해서 확인해보자!
```
